# Changelog

All notable changes to this project will be documented in this file.

## [v0.2.0] - 2026-01-31

### 🎉 重大更新：图形用户界面和权限管理

#### 新增功能
- **图形用户界面 (GUI)**
  - 完整的Tkinter图形界面
  - 三种操作模式：安全/标准/激进
  - 实时命令行输出显示
  - 进度条和状态指示
  - 示例软件名称填充

- **权限管理系统**
  - 自动权限检查功能
  - 平台特定的权限提示
  - 清晰的错误信息和解决方案
  - 安全模式无需权限

#### 改进功能
- **命令行工具增强**
  - 添加权限检查 (`check_permissions()`)
  - 改进错误处理和用户提示
  - 提前检查并提示需要sudo权限
  - 更好的交互体验

- **用户体验优化**
  - GUI显示当前权限状态
  - 详细的权限警告对话框
  - 改进的错误处理流程
  - 平台特定的解决方案提示

#### 文档更新
- 添加GUI使用指南 (`GUI_README.md`)
- 更新主README添加权限要求说明
- 详细的使用示例文档
- 故障排除指南

#### 新增文件
- `gui.py` - GUI主程序
- `run_gui.py` - GUI启动器（含依赖检查）
- `GUI_README.md` - GUI使用指南
- `test_fix.py` - 权限测试工具

#### 技术改进
- 线程安全的实时输出捕获
- 响应式GUI布局设计
- 跨平台兼容性增强
- 与现有命令行工具无缝集成

### 🔧 修复问题
- **权限问题**：解决用户反馈的"卸载不了"问题
- **错误处理**：改进权限不足时的错误提示
- **用户体验**：提供更清晰的指导和解决方案
- **文档完整性**：补充缺失的使用说明

### 📋 使用方式更新

#### 命令行版本
```bash
# 安全模式（无需权限）
python3 main.py "软件名" --safe

# 卸载模式（需要sudo）
sudo python3 main.py "软件名"
sudo python3 main.py "软件名" --aggressive
```

#### GUI版本
```bash
# 需要sudo权限运行GUI
sudo python3 run_gui.py
# 或
python3 run_gui.py  # 会提示权限问题
```

### 🎯 系统要求
- **Python**: 3.7+
- **Tkinter**: GUI依赖（自动检查）
- **psutil**: 进程管理（自动安装）
- **权限**: 卸载操作需要管理员权限

### 🔄 向后兼容性
- 完全兼容v0.1.0命令行接口
- 新增功能不影响现有使用方式
- 配置文件格式保持不变
